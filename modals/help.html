Help Left Menubar You can see list of chapters and quickly change between chapters using the book icon on the left menubar Question icon shows this help screen Gears icon opens settings panel Information icon shows about screen Controls for scrolling chapter content You can scroll chapter content using keyboard keys: up arrow, down arrow, page down, page up, home and end keys or with mouse wheel. Bottom navigation At the bottom of the book content, you can navigate through chapter's sub sections. If you scroll book content, an arrow button appears to right bottom which you can use to scroll book chapter content to top
